# Serwis S.O.M. Landing

Landing page for auto service based in Poznań.

## Tech stack

- React
- TypeScript
- Vite
- SCSS

## Features

- Responsive layout (mobile-first)
- Contact options:
  - Phone call (primary CTA)
  - Telegram bot
  - Contact form
- Clean UI focused on conversion

## Project structure
```
src/

│ └── styles/

│ └── icons/
├── components/
│ └── ui/
├── sections/
├── utils/
├── types/
├── constants/
├── App.tsx
└── main.tsx
```

## Getting started

Install dependencies:

```npm install```

Run development server:

```npm run dev```

Build project:

```npm run build```
